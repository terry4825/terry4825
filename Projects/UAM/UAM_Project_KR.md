# 🛸 UAM 최적 경로 생성 시뮬레이션 및 GUI 개발

## 📌 프로젝트 개요
* **기간:** 2023.07 - 2023.12 (인턴십 프로젝트)
* **목표:** 강화학습(Q-Learning)을 활용한 도심항공모빌리티(UAM) 최적 비행 경로 생성 및 실시간 가시화 시스템 구축
* **주요 역할:** C++ 기반 강화학습 알고리즘 구현 및 Qt 프레임워크를 이용한 시뮬레이터 GUI 설계

## 🛠 사용 기술
* **언어:** C++
* **프레임워크:** Qt (GUI 개발)
* **알고리즘:** Reinforcement Learning (Q-Learning)
* **도구:** Visual Studio

## 🚀 핵심 성과 및 기능
* **최적 경로 알고리즘:** Q-Learning을 적용하여 동적 장애물을 회피하고 최단 거리를 도출하는 로직 설계
* **사용자 인터페이스(GUI):** Qt를 사용하여 비행 경로, 장애물, 환경 요소를 실시간으로 모니터링할 수 있는 대시보드 구현
* **연산 효율성 극대화:** 로컬 최적화 기법을 도입하여 경로 생성 연산 시간을 **3.87초**로 단축
* **공인 기관 신뢰성 검증 (KTL):** KTL(한국산업기술시험원)의 검수를 통해 알고리즘의 정합성 및 안정성 공식 인증
    * 총 50회의 시뮬레이션 테스트 중 48회 성공(96%)을 기록하며 실용화 가능성 입증

## 🧠 기술적 도전 및 해결: 로컬 최적화 (Local Optimization)
* **문제:** $100\text{km} \times 100\text{km}$ ($100 \times 100$ Grid)의 광범위한 환경을 전역적으로 탐색할 경우, 연산 복잡도가 너무 높아 실시간 경로 생성이 불가능했음.
* **해결:** 현재 기체 위치를 중심으로 $20 \times 20$ 영역($10\text{km}$ 반경)만을 집중 탐색하는 **로컬 최적화** 기법 적용.
* **결과:** 전체 맵 대비 탐색 공간을 **96% 절감**하여 데이터 처리 효율을 극대화했으며, 이를 통해 실시간 시뮬레이션 환경을 성공적으로 구축함.
