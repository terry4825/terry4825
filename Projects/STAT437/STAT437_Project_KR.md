# ❄️ 미국 주별 코카인 오남용 패턴 분석 및 군집화
> **비지도 학습(K-medoids, Fuzzy C-means)을 통한 보건 데이터의 구조적 특성 진단 및 분석**

## 📌 프로젝트 개요
* **수행 기간:** 2024.03.04 - 2024.05.13
* **성격:** 공중보건 데이터 기반 비지도 학습(Unsupervised Learning) 프로젝트
* **목표:** 미국 50개 주 및 DC의 연령대별 코카인 사용률 데이터를 분석하여 유의미한 군집(Cluster) 식별 및 데이터 구조 진단
* **핵심 역할:** **분석 모델 선정 및 데이터 타당성 검증**
    * 데이터의 이상치(Outlier)와 모호한 경계를 처리하기 위해 **K-medoids** 및 **Fuzzy C-means** 모델을 직접 선정하여 적용
    * 분석 결과에 대한 통계적 사후 검토를 통해 데이터셋의 구조적 한계점 도출 및 개선 방향 제시

## 🛠 분석 기술 스택
* **Language:** R
* **Selected Algorithms:** * **K-medoids (PAM):** 실제 데이터 포인트를 중심점으로 사용하여 이상치에 강건한(Robust) 군집화 시도
    * **Fuzzy C-means:** 데이터 포인트가 여러 군집에 속할 수 있는 가중치를 분석하여 보건 데이터의 모호성 탐색
* **Methods:** Silhouette Coefficient (실루엣 계수), Hierarchical Clustering (Single Linkage), Data Preprocessing

## 🚀 주요 수행 내용 및 기술적 도전

### 1. 데이터 전처리 및 탐색적 분석 (EDA)
* 17년간의 미국 전역 코카인 사용률 데이터를 분석 가능한 형태로 구조화.
* 결측치가 없음을 확인하고, 인구수(Population) 등 수치형 변수의 왜곡(Skewness)이 모델에 미치는 잠재적 영향 파악.

### 2. 분석 모델 적용 및 한계 진단 (Technical Insights)
* **스케일 왜곡 포착:** 인구수 데이터의 스케일이 너무 커서 초기 군집화 결과가 인구 규모에 편향되는 현상을 발견, 변수 선택과 정규화의 중요성 역설.
* **시계열적 유사성 확인:** '주(State)'와 '연도(Year)' 변수의 특성으로 인해 동일 지역 데이터가 과도하게 밀집(예: 모든 연도의 California 데이터가 하나로 묶임)되는 현상을 분석하여 데이터 재구조화 필요성 도출.
* **모델 검증:** 실루엣 계수 및 덴드로그램 분석 결과, 현재 데이터 구조에서는 군집 간 경계가 통계적으로 명확하지 않음(Inconclusive)을 과학적으로 증명.

### 3. 향후 분석 고도화 제안 (Conclusion 기반)
* 단순 사용률 데이터의 한계를 극복하기 위해 '합성 오피오이드(펜타닐 등) 오남용 데이터'와의 다변량 연계 분석이 필요하다는 기술적 로드맵 제시.

## 🏆 주요 성과 및 인사이트
* **비판적 분석 사고:** 분석 결과를 무조건 수용하지 않고, 모델이 유의미한 결론을 내지 못한 원인을 데이터의 구조적 결함(Skewness, Auto-correlation)에서 찾아낸 분석적 통찰력 발휘.
* **정직한 데이터 과학:** 통계적 가설이 성립하지 않는 상황을 논리적으로 설명하고, 이를 바탕으로 더 나은 분석 모델을 위한 데이터 설계(Design) 방향을 제시하는 전문성 확보.
