# ❄️ 미국 주별 코카인 오남용 패턴 분석 및 군집화
> **비지도 학습(K-medoids, Hierarchical Clustering)을 활용한 보건 데이터 패턴 식별 프로젝트**

## 📌 프로젝트 개요
* **수행 기간:** 2024.03.04 - 2024.05.13
* **성격:** 공중보건 데이터 기반 통계 분석 프로젝트
* **목표:** 미국 50개 주 및 DC의 연령대별 코카인 사용률 데이터를 분석하여 유사한 오남용 패턴을 가진 그룹을 분류하고 정책적 시사점 도출
* **핵심 역할:** **데이터 분석 모델 선정 및 알고리즘 적용**
    * 프로젝트 목적에 부합하는 최적의 비지도 학습 모델 2종(K-medoids, 계층적 군집화)을 직접 선정
    * 선정된 모델을 R을 활용해 실제 데이터에 적용하고 분석 결과 도출 및 모델 검증 수행

## 🛠 분석 기술 스택
* **Language:** R
* **Selected Algorithms:** * **K-medoids (PAM):** 실제 데이터 포인트를 중심점으로 사용해 이상치(Outlier)에 강건한 군집화 수행
    * **Hierarchical Clustering:** AGNES 알고리즘 및 다양한 연결법(Single, Complete, Average, Ward) 적용 및 비교
* **Evaluation:** Silhouette Coefficient (실루엣 계수), Dendrogram Analysis

## 🚀 주요 수행 내용

### 1. 분석 모델 선정 및 타당성 검토
* 데이터의 특성(주별/연령별 사용률)을 고려하여, 일반적인 K-means보다 이상치에 강한 **K-medoids**와 데이터 간의 계층적 구조를 확인할 수 있는 **계층적 군집화**를 분석 방법론으로 최종 채택.

### 2. 알고리즘 적용 및 최적화
* **계층적 군집화:** Single, Complete, Average, Ward 연결법을 각각 적용하여 데이터의 응집도를 비교하고, 가장 명확한 군집 구조를 보이는 모델 선택.
* **K-medoids (PAM):** 실루엣 플롯(Silhouette Plot)을 통해 군집의 품질을 정량적으로 평가하고 최적의 군집 수($k$)를 결정하여 분석의 객관성 확보.

### 3. 데이터 패턴 해석
* 17년간의 장기 데이터를 분석하여 코카인 오남용 위험이 높은 주(State)들의 공통적인 인구통계학적 특성을 도출.
* 분석된 클러스터 정보를 바탕으로 효율적인 보건 자원 배분을 위한 전략적 가이드라인 제시.

## 🏆 주요 성과 및 인사이트
* **독립적 모델링 역량:** 문제 해결을 위해 스스로 통계 모델을 선정하고, 이를 데이터에 성공적으로 적용하여 유의미한 결론을 도출하는 분석 설계 능력 증명.
* **통계적 정밀성:** 단순히 알고리즘을 구동하는 것에 그치지 않고, 다양한 파라미터 비교와 실루엣 계수 검증을 통해 분석 결과의 신뢰도를 극대화함.
