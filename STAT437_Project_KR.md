# ❄️ 미국 주별 코카인 오남용 패턴 분석 및 군집화 (STAT 437)
> **비지도 학습(K-medoids, Hierarchical Clustering)을 활용한 보건 데이터 패턴 식별 프로젝트**

## 📌 프로젝트 개요
* **수행 기간:** 2024.08 - 2024.12 (보고서 최종 수정: 2025.03)
* **성격:** UIUC STAT 437 (Nonparametric Statistics/Multivariate Analysis) 팀 프로젝트
* **목표:** 미국 50개 주 및 DC의 연령대별 코카인 사용률 데이터를 분석하여 유사한 오남용 패턴을 가진 주들을 군집화하고 공중보건 정책적 시사점 도출
* **사용 도구:** R, RStudio

## 🛠 분석 기술 스택
* **Language:** R
* **Algorithms:** * **K-medoids (PAM):** 이상치(Outlier)에 강건한 군집화 수행
  * **Hierarchical Clustering:** AGNES (Agglomerative Nesting) 및 다양한 연결법(Single, Complete, Average, Ward) 비교
* **Evaluation:** Silhouette Coefficient (실루엣 계수), Dendrogram Analysis
* **Libraries:** `cluster`, `factoextra`, `tidyverse`

## 🚀 주요 분석 내용

### 1. 고차원 데이터 전처리 및 탐색적 분석 (EDA)
* 17년간 축적된 미국 전역의 연령대별(12-17, 18-25, 26+) 코카인 사용률 데이터를 분석에 적합한 형태로 구조화.
* 주별 사용 트렌드와 연령 그룹 간의 상관관계를 파악하여 군집화의 타당성 확보.

### 2. 최적 군집 모델 설계 및 검증
* **계층적 군집화:** Single, Complete, Average, Ward의 4가지 연결법을 비교 분석. Single Linkage에서 나타나는 체이닝 현상을 확인하고, 데이터 구조를 가장 잘 반영하는 알고리즘 선택.
* **K-medoids (PAM):** K-means의 한계를 보완하기 위해 실제 데이터 포인트를 중심점으로 사용하는 PAM 알고리즘 적용. 
* **모델 평가:** 실루엣 플롯(Silhouette Plot)을 통해 군집의 응집도와 분리도를 정량적으로 평가하여 최적의 군집 수($k$) 결정.

### 3. 분석 결과 해석 및 정책적 제안
* 코카인 오남용 문제가 심각한 특정 클러스터를 식별하고, 해당 주들의 지리적/인구통계적 공통점 분석.
* 펜타닐 등 합성 오피오이드 오남용 데이터와의 연계 분석 필요성을 제기하며 보건 당국의 효율적인 자원 배분 전략 제안.

## 🏆 주요 성과 및 인사이트
* **비지도 학습 역량:** 라벨이 없는 복잡한 보건 데이터에서 유의미한 패턴을 스스로 찾아내는 통계적 모델링 능력 입증.
* **모델 검증의 정밀성:** 단순히 결과만 도출하는 것이 아니라, 다양한 거리 측정 기준과 연결법을 비교 검토하며 분석의 신뢰성을 높임.
* **실무적 가치 창출:** 통계 분석 결과를 정책 입안자가 활용할 수 있는 형태의 인사이트로 변환하는 '데이터 스토리텔링' 경험 확보.
