# ⚖️ AI 기반 법률 개선 및 데이터 활용 솔루션 제안

## 📌 프로젝트 개요
* **기간:** 2023.07 - 2023.09 (공모전 참여)
* **주최:** 법제처 (제1회 법률데이터 활용 아이디어 공모전)
* **목표:** 방대한 법률 데이터를 효율적으로 분석하여 법령 개선점을 도출하는 AI 모델 아키텍처 기획
* **주요 역할:** 데이터 전처리 전략 수립 및 알고리즘(S-BERT, XGBoost) 선정 및 파이프라인 설계

## 🛠 제안 기술 스택 (Proposed Tech Stack)
* **Natural Language Processing:** Sentence-BERT (S-BERT)
* **Machine Learning:** XGBoost, Random Forest
* **Analysis Tools:** Python, Scikit-learn
* **Data Focus:** 법령 텍스트 데이터 및 판례 데이터

## 🚀 핵심 기획 내용
* **문맥 기반 법령 검색:** Sentence-BERT를 활용하여 단순 키워드 매칭이 아닌, 법령의 의미적 유사성을 바탕으로 관련 조문을 추출하는 메커니즘 설계
* **법령 개선점 분류 모델:** XGBoost와 Random Forest를 결합한 앙상블 모델을 통해 개정이 필요한 법령의 우선순위와 카테고리를 자동 분류하는 로직 제안
* **데이터 파이프라인:** 비정형 법률 텍스트를 벡터화하여 정형 데이터와 결합하는 분석 워크플로우 구체화

## 🏆 주요 성과
* **논리성 검증:** 기술적 타당성과 혁신성을 인정받아 **제1회 법률데이터 활용 아이디어 공모전 1차 서류 심사 통과**
* **통계적 접근:** UIUC에서 학습한 통계적 가설 검정 사고를 데이터 모델링에 접목하여 분석의 신뢰성 확보

## 🧠 인사이트
* 법률 도메인의 특성상 텍스트의 미묘한 차이가 큰 법적 해석의 차이를 만든다는 점을 파악하고, 이를 해결하기 위해 고도화된 임베딩 기술(S-BERT)의 필요성을 논리적으로 증명함
