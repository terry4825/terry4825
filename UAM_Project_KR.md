# UAM 최적 경로 생성 시뮬레이션 및 GUI 개발

## 📌 프로젝트 개요
* **기간:** 2025.07 - 2025.12 (인턴십 프로젝트)
* **목표:** 강화학습(Q-Learning)을 활용한 UAM 최적 비행 경로 생성 및 가시화 도구 개발
* **주요 역할:** C++ 기반 시뮬레이션 로직 구현 및 Qt 프레임워크를 이용한 GUI 설계

## 🛠 사용 기술
* **언어:** C++
* **프레임워크:** Qt (GUI 개발)
* **알고리즘:** Reinforcement Learning (Q-Learning)
* **도구:** Visual Studio

## 🚀 핵심 성과 및 기능
* **최적 경로 알고리즘:** Q-Learning을 적용하여 동적 장애물을 회피하고 최단 거리를 도출하는 로직 설계
* **사용자 인터페이스(GUI):** Qt 프레임워크를 사용하여 비행 경로와 환경 요소를 실시간으로 확인할 수 있는 대시보드 구현
* **연산 효율성:** 로컬 최적화 기법을 도입하여 경로 생성 연산 시간을 **3.87초**로 단축
* **신뢰성 검증:** 총 50회의 시뮬레이션 테스트 중 **48회 성공(96%)**을 기록하며 알고리즘의 안정성 입증

## 🧠 기술적 도전 및 해결
* **문제:** 복잡한 알고리즘 결과를 텍스트 로그로만 확인하기 어려워 데이터 직관성이 떨어짐
* **해결:** Qt를 활용한 GUI 시뮬레이터를 직접 개발하여, 경로 생성 과정을 시각화함으로써 알고리즘의 동작 상태를 즉각적으로 모니터링할 수 있도록 개선
