# UAM 최적 경로 생성 시뮬레이션 개발

## 📌 프로젝트 개요
* **기간:** 2025.07 - 2025.12 (인턴십 프로젝트)
* **목표:** 강화학습(Q-Learning)을 활용하여 도심항공모빌리티(UAM)의 안전하고 효율적인 최적 비행 경로 생성
* **주요 역할:** C++ 기반 시뮬레이터 환경 구축 및 경로 최적화 알고리즘 구현

## 🛠 사용 기술
* **언어:** C++
* **알고리즘:** Reinforcement Learning (Q-Learning)
* **도구:** Git, Visual Studio

## 🚀 핵심 성과 및 기능
* **최적 경로 알고리즘:** Q-Learning을 적용하여 장애물을 회피하고 최단 거리를 도출하는 로직 설계
* **연산 효율성:** 로컬 최적화 기법을 도입하여 경로 생성 연산 시간을 **3.87초**로 단축
* **신뢰성 검증:** 총 50회의 테스트 중 **48회 성공(96%)**을 기록하며 알고리즘의 안정성 입증
* **객체 지향 설계:** 비행체, 장애물, 환경 요소를 클래스화하여 확장성 있는 시뮬레이션 구조 설계

## 🧠 기술적 도전 및 해결
* **문제:** 초기 알고리즘의 높은 연산 복잡도로 인해 실시간 경로 생성에 한계가 있었음
* **해결:** 탐색 범위를 전략적으로 제한하는 로컬 최적화 방식을 도입하여 정확도를 유지하면서 연산 속도를 대폭 개선
